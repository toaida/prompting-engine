## PHILOSOPHY

**Core Problem:** AI-generated images show clothing as flat textures placed on bodies — no understanding of why fabric hangs, pools, drapes, or clings. Clothing exists in 3D space and responds to gravity, body posture, and layer interaction.

**The Fix:** Describe WHY clothing looks the way it does — physics vocabulary that explains garment behavior. NOT "she's wearing an open cardigan" but "the cardigan's weight pulls it open, fabric falling away from shoulders under its own weight."

## FABRIC PHYSICS VOCABULARY

### Gravity Effects

| Mechanic | Description | Visual Result |
|----------|-------------|---------------|
| Weight pull | Fabric sinks under own weight | Drape away from body, elongation |
| Pooling | Fabric accumulates at lowest point | Gathering at hem, bunching |
| Compression | Gravity + surface = flattening | Fabric thins, clings tight |
| Stretching | Weight pulls, elasticity resists | Tension lines, transparency |
| Sliding | Gravity overcomes friction | Fabric slipping, repositioning |

### Body Interaction

| Mechanic | Description | Visual Result |
|----------|-------------|---------------|
| Contouring | Fabric follows body curve | Silhouette visible through fabric |
| Gap creation | Body moves, fabric stays | Space between fabric and skin |
| Pressure point | Body contacts surface | Fabric compression at contact |
| Release | Pressure removed | Fabric expanding, returning |
| Tension line | Pull from two directions | Fabric taut, lines radiating |

### Layer Behavior

| Mechanic | Description | Visual Result |
|----------|-------------|---------------|
| Stacking | Upper layer weight on lower | Compression, creasing |
| Exposure | Upper layer parts, reveals lower | Neckline gap, hem reveal |
| Shadowing | Upper layer casts shadow | Lower layer darker |
| Movement lag | Lower layer moves less | Different fabric behavior speeds |

### 1. OPEN CARDIGAN DRAPE

**Physics Logic:**
```
Cardigan is heavy → gravity pulls down → front panels separate → 
fabric falls away from torso → inner garment visible through opening → 
shoulder seam visible → sleeves hang with less body contact
```

## GARMENT PHYSICS INTEGRATION RULES

### Rule 1: Explain the WHY

```
BAD: "Open cardigan, visible shirt"
GOOD: "Cardigan hanging open from own weight, inner shirt visible through gap"
```

### Rule 2: Use Physics Language

- Gravity pulls, pools, compresses
- Fabric falls, drapes, clings
- Body creates pressure, contact, gap
- Layers stack, expose, shadow

### Rule 3: Connect to Pose

Garment behavior emerges from pose:
- Sitting → skirt displacement
- Reaching → sleeve slippage  
- Turning → fabric separation
- Standing → pooling at hem

### Rule 4: Include Weight

Clothing has weight. Heavier fabrics behave differently than light fabrics:
- Heavy knit → less movement, more drape
- Light chiffon → more movement, more cling
- Elastic → compression, contouring

### Rule 5: Avoid Fetish Framing

**This is adult fashion realism, not fetish content.**

```
BAD: "clinging wet fabric emphasizing curves"
GOOD: "wet fabric heavier, clinging to body contour from rain"

BAD: "sheer fabric showing body"
GOOD: "thin fabric following body curve, skin tone visible through fabric"
```
