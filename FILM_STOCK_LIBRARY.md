## FUJIFILM PRO 400H (Discontinued 2012)

### The Signature Japanese Look

**Color Science:**
```
HIGHLIGHTS: Slightly warm with yellow push
SKIN: Peach-cream undertone, never orange
SHADOWS: Cool magenta shift, never blue
GREEN: Slightly desaturated, olive undertone
WHITE: Cream, not pure
```

**Technical Characteristics:**
```
GRAIN: Very fine, tight, barely visible at portrait distances
HALATION: Soft, spread into highlights
DYNAMIC RANGE: High, shadows roll off gently
SKIN RENDERING: Signature "glow" — slight light scatter in highlights
CONTRAST: Medium-low, never harsh
```

**When to Use:**
- Cloudy/overcast light (enhances the peach skin)
- Indoor window light (enhances the glow)
- Skin-forward compositions (maximizes the peach)
- Late Heisei era aesthetic (2000s-2010s)
- Gravure idol aesthetic

**Prompt Vocabulary:**
```
"Pro 400H color science: peach skin undertone, warm highlight rolloff, 
cool shadow with magenta shift, cream whites, desaturated greens, 
fine grain, soft halo halation, medium-low contrast, skin glow"
```

## JAPANESE GRAVURE PHOTOBOOK RENDERING

### Late Heisei (1990s-2019) Visual Language

**Core Characteristics:**
```
SKIN: Soft focus on skin, slight blur, creamy
EYES: Clear focus, emphasized lashes
HAIR: Slightly darker than reality, soft
LIGHT: Soft diffused window light dominant
COMPOSITION: Centered, face-forward, intimate
OVERALL: Dreamy, soft, romantic, not clinical
```

**Technical Notes:**
```
LENS: 85mm-135mm portrait primes
APERTURE: Wide (f/1.4-f/2.8) for shallow DOF
FOCUS: Eyes always sharp, skin soft
BAYER: Slight color fringing in corners
POST: Softness, skin smoothing, warm tint
```

**Specific Look Elements:**
- **Skin:** Porcelain quality, slightly airbrushed, never harsh
- **Eyes:** Clear iris detail, emphasized, slightly brightened
- **Lips:** Slightly pinker than natural, soft highlight
- **Cheeks:** Gentle rosy blush from light, not makeup
- **Nose:** Soft, shadow never harsh
- **Background:** Strong bokeh or pure white/cream

**When to Use:**
- Gravure aesthetic
- Idol photography style
- Romantic intimate moments
- Beauty-forward compositions
- Hotel/indoor intimate scenes

**Prompt Vocabulary:**
```
"Gravure photobook: soft focus skin, creamy porcelain quality, 
sharp clear eyes with emphasized lashes, soft diffused window light, 
centered face-forward composition, dreamy romantic tone, 
slight warm tint, gentle bokeh background, beauty-forward"
```

## SPECIFIC TECHNICAL TERMS

### Grain Patterns

| Film | Grain Character | Texture |
|------|-----------------|---------|
| Pro 400H | Very fine, tight | Creamy |
| C200 | Fine, visible | Organic |
| Gold 200 | Medium, visible | Warm |
| Portra 400 | Very fine, smooth | Clean |

**Prompt for grain:**
```
"[FILM] grain: [fine/medium/visible], [tight/organic/warm] texture, 
[creamy/organic/clean] feel"
```

### Halation

**What it is:** Light blooming around bright areas (sun, highlights)

| Film | Halation Character |
|------|-------------------|
| Pro 400H | Soft, warm orange, gentle spread |
| Gold 200 | Orange-amber, medium spread |
| CineStill 800T | Orange glow (removable remjet) |

**Prompt for halation:**
```
"Soft halation around bright areas, warm orange spread, 
gentle glow into shadow areas"
```

### Highlight Roll-off

**What it is:** How highlights transition to blown areas

| Film | Roll-off Character |
|------|-------------------|
| Pro 400H | Gentle, creamy, never harsh |
| Portra 400 | Smooth, wide latitude |
| Gold 200 | Warm, slightly punchy |
| C200 | Moderate, consumer-friendly |

### Shadow Behavior

| Film | Shadow Character |
|------|-----------------|
| Pro 400H | Cool magenta, soft |
| Portra 400 | Neutral-cyan, clean |
| Gold 200 | Warm brown, rich |
| C200 | Slight green-teal |

## FILM STOCK INTEGRATION RULES

### Rule 1: Match Film to Lighting Condition

```
OVERCAST: Pro 400H (enhances peach skin)
GOLDEN HOUR: Gold 200 (maximizes warmth)
INDOOR WINDOW: Pro 400H (window light ideal)
TUNGSTEN NIGHT: CineStill 800T (orange/blue war)
```

### Rule 2: Include Technical Specifics

```
BAD: "Film look"
GOOD: "Pro 400H: peach skin undertone, cool magenta shadows, 
       fine tight grain, soft halation, medium-low contrast"
```

### Rule 3: Match Film to Era/Mood

```
90s NOSTALGIA: Superia Premium
2000s CASUAL: C200
2000s DREAMY: Pro 400H
MODERN: Portra 400
GRAVURE: Pro 400H + soft focus
```

### Rule 4: Combine with Spatial + Existence

```
FILM: Pro 400H + EXISTENCE: hotel mirror check + SPATIAL: foreground perfume bottle
→ Gravure photobook aesthetic + believable photo reason + spatial depth
```

### Rule 5: Anti-AI Specific Corrections

**Pro 400H anti-AI markers:**
- Cool shadow magenta (stops blue-correction tendency)
- Peach skin (stops orange-correction tendency)
- Soft halation (stops clean highlight tendency)
- Fine grain (stops perfect-smooth tendency)

**C200 anti-AI markers:**
- Slight color fade (stops over-saturated tendency)
- Muted green (stops punchy-green tendency)
- Casual tone (stops professional-perfect tendency)
