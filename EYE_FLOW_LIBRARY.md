# EYE FLOW LIBRARY
### V17 — Visual Reading-Path Templates
### Status: ACTIVE — Runtime System

---

## PHILOSOPHY

**Core Problem:** AI-generated images often lack intentional visual flow — the viewer's eye doesn't know where to go, or goes to the wrong place first. Real photography guides the eye deliberately through the frame.

**The Fix:** Build visual reading paths that take the viewer through the image in a specific order. Eye flow is a guided journey, not a random landing.

---

## EYE FLOW VOCABULARY

### Reading Direction Terms

| Term | Meaning |
|------|---------|
| ANCHOR | First thing the eye finds (usually brightest or highest contrast) |
| PATH | How eye moves through frame |
| DESTINATION | Final resting point (usually face/eyes) |
| SUPPORT | Secondary elements that guide toward destination |

### Flow Manipulation Terms

| Term | Meaning |
|------|---------|
| LEAD | Element that leads eye to subject |
| FRAME | Element that frames and directs attention |
| HOLD | Element that pauses eye before moving |
| RELEASE | Element that moves eye to next zone |

## EYE FLOW TEMPLATES

### Template 1: BOTTOM-UP

**Path:** Floor/Object → Body → Face

**Used for:**
- Ground-level shots
- Documentary style
- Environmental immersion

**Example: Beach Towel**
```
ANCHOR: Sand grain detail (bottom edge)
↓ PATH: Towel fold leading upward
→ BODY: Subject's back/shoulder
→ DESTINATION: Face/eyes

Prompt: "Camera low at sand level, eye travels up through 
towel fold line to body to face, ground visible at frame edge"
```

**Example: MTR Standing**
```
ANCHOR: Yellow safety line (floor level)
↓ PATH: Pole leading upward
→ BODY: Subject's torso and arm on pole
→ DESTINATION: Face looking down at phone

Prompt: "Eye travels from floor line up pole to hand to face, 
ground element anchors start"
```

## EYE FLOW INTEGRATION RULES

### Rule 1: Start with Anchor, End with Destination

```
EVERY PROMPT NEEDS:
ANCHOR → [brightest or highest contrast element]
PATH → [leading line to subject]
DESTINATION → [face/eyes as final point]
```

### Rule 2: Foreground Creates Depth

```
Foreground anchor = strongest depth cue
5-15% frame foreground object
Eye moves foreground → mid → background
```

### Rule 3: Light Guides Eye

```
Brightest area = eye's first landing
Use window light, screen glow, lamp
Eye moves from light to lit subject
```

### Rule 4: Create One Clear Path

```
AVOID: Multiple competing anchors
GOOD: One dominant path through frame
Guide, don't confuse
```

### Rule 5: Face = Final Destination

```
Almost always: face/eyes are final resting point
Exception: activity-focused where hands/result are final
Rule: If face visible, face is destination
```

## EYE FLOW + OTHER SYSTEMS

### Eye Flow + Spatial Composition

```
SPATIAL: Foreground → Subject → Background
EYE: Anchor → Path → Destination

COMBINED:
Foreground object (anchor) + mid-ground subject + background 
creates BOTH spatial depth AND eye path
```

### Eye Flow + Film Stock

```
EYE: Light source → Subject → Face
FILM: Pro 400H (soft glow) = dreamy eye path
FILM: Gold 200 (warm) = nostalgic eye path
FILM: C200 (casual) = documentary eye path
```

### Eye Flow + Existence Logic

```
EYE: Activity object → Hands → Face
EXISTENCE: Why photo exists

COMPOUND:
"Phone in hand (anchor) → checking message → face showing 
reaction → because she's texting boyfriend"
```

## GRAVURE-SPECIFIC EYE FLOW

### Beauty-Forward Gravure Path

**Path:** Hair → Face → Eyes

**Used for:**
- Gravure aesthetic
- Portrait emphasis
- Beauty content

**Example:**
```
ANCHOR: Hair framing face (top of frame)
→ PATH: Hair line leading down
→ BODY: Face in center
→ DESTINATION: Eyes (clear focus)

Prompt: "Eye starts at hair framing top of face, travels 
down to face center to clear eyes, centered beauty composition"
```

### Soft Focus Gravure Path

**Path:** Body contour → Face → Eyes

**Used for:**
- Romantic mood
- Dreamy atmosphere
- Intimate aesthetic

**Example:**
```
ANCHOR: Soft body curve (forehead or shoulder)
→ PATH: Light fall on body leading to face
→ BODY: Face soft focus, body softer
→ DESTINATION: Eyes catching light

Prompt: "Soft focus body edge leads to face, eye travels 
through soft blur to sharp eyes, dreamy romantic path"
```

## QUICK GENERATION RULE

**Every image needs ONE eye flow path:**
```
ASK: Where does the eye start? Where does it end?

START: Brightest element, foreground, corner, floor
MIDDLE: Leading line (path)
END: Face/eyes (almost always)

If face visible: face is destination
If no clear path: add foreground anchor
```
