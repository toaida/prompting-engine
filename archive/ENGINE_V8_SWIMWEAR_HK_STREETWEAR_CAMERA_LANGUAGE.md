# V8 New Engine Modules

## 1. SWIMWEAR_COVER_LAYER_SYSTEM

Swimwear becomes IG-safe and socially believable when it is embedded in lifestyle layering.

Use:

- loose linen shirt over one-piece
- sarong over high-waist swim bottom
- oversized windbreaker over bikini top
- mesh long-sleeve over swim set
- baby tee with low-rise swim bottom, adult-coded only
- knit resort cardigan over one-piece
- oversized shirt + towel + beach tote
- swimwear under resort shorts
- swimsuit + cap + sunglasses + sandals

Core rule:

The cover layer must have physical behavior:
- drape
- wrinkled hem
- salt-air crinkle
- sleeve roll
- wind movement
- humidity-softened folds
- loose knot
- fabric shift from walking/sitting

Avoid:
- swimwear catalogue stiffness
- explicit wet-cling language
- shower-corner framing
- private bedroom swimwear logic

---

## 2. SAFE_BOLDNESS_CALIBRATION_SYSTEM

V8 boldness is controlled through context.

Safe boldness increases when:

- public or semi-public setting exists
- swimwear is justified by beach / pool / boat / resort
- cover layer softens exposure
- background adults / social context are visible
- camera keeps face/environment important
- outfit is described as styling, not access

Unsafe boldness increases when:

- private setting + swimwear
- direct seduction language
- explicit body-part priority
- crotch-forward pose
- wet fabric cling emphasis
- low camera aimed upward at swimsuit
- no social reason for photo

Core instruction:

Bold but platform-safe means:
more confidence, not more explicitness.

---

## 3. IG_SAFE_RESORT_CAMERA_SYSTEM

Preferred swimwear camera types:

- friend-shot beach candid
- resort diary smartphone shot
- poolside telephoto observation
- mirrorless travel portrait
- compact digital pool-night flash
- boat deck wind candid
- beach towel lifestyle frame
- standing waterline portrait
- sitting poolside with environment visible
- over-shoulder beach walk
- environment-heavy seascape with subject smaller in frame

Camera rules:

- keep face, environment, or activity as visual anchor
- avoid crotch-centered crops
- avoid low-angle swimsuit emphasis
- use body language as lifestyle behavior
- include foreground towels, bags, water, chairs, railings, drinks, sandals

## 4. SWIMWEAR_POSTURE_NATURALISM_SYSTEM

Safe swimwear actions:

- adjusting sunglasses
- holding beach tote
- brushing windblown hair back
- stepping out of shallow water
- wrapping linen shirt around shoulders
- sitting on towel while checking phone
- walking along pier
- leaning on boat railing
- holding iced drink near pool
- laughing while wind moves shirt
- putting sandals back on
- tying sarong
- carrying towel under arm

Prompt logic:

activity → posture → fabric response → camera framing → emotional tone

Never:

body display → pose → camera justification

## 5. HK_STREETWEAR_LAYERING_SYSTEM

Hong Kong streetwear realism comes from humidity, MTR commuting, mall AC, and middle-class fashion logic.

Useful outfit families:

- oversized tee + fitted shorts
- fitted knit top + cargo skirt
- tank top + loose shirt + mini skirt
- adult-coded baby tee + low-rise cargos
- sporty zip top + non-school mini skirt
- windbreaker + fitted crop + track shorts
- linen shirt + denim shorts
- mesh long sleeve over fitted layer
- utility vest + skirt / shorts
- off-duty office top + sneakers
- black-on-black nightlife casual
- rain jacket + fitted base layer
- athleisure yoga pants + oversized jacket

Core contrast:

oversized element + fitted element + functional accessory

## 8. FABRIC_BEHAVIOR_ENGINE

Use fabric to show body indirectly.

Swimwear fabric vocabulary:

- matte swim fabric
- ribbed swim texture
- gathered swim construction
- sun-dried fabric
- salt-air texture
- pool-light reflections
- towel contact folds
- linen shirt drape
- sarong knot folds
- mesh cover-up texture
- wind-lifted cover layer

Streetwear fabric vocabulary:

- cotton tee collapse
- nylon windbreaker crinkle
- cargo pocket weight
- denim hem stiffness
- ribbed knit stretch
- mesh layer transparency
- rain jacket sheen
- hoodie cuff stretch
- skirt hem swing
- tote strap pull

Avoid:

- anatomy-first language
- chest physics
- explicit cling
- overexplained compression

## 9. STREETWEAR_POSTURE_BEHAVIOR_SYSTEM

# V8 Prompt Construction Order

For swimwear:

adult identity lock
→ location justification
→ social-photo reason
→ activity
→ cover layer
→ swimwear fabric
→ posture mechanics
→ camera angle
→ lighting / water / wind
→ safe boldness constraints

For HK streetwear:

adult identity lock
→ HK local environment
→ reason photo exists
→ walking/waiting/social activity
→ outfit architecture
→ bag/shoe/accessory logic
→ posture and hand-object behavior
→ camera style
→ local texture
→ anti-AI imperfection

---

# V8 Batch Rule

Across a 10-prompt batch, rotate:

- swimwear vs streetwear
- public vs semi-public
- beach vs pool vs pier vs boat
- MTR vs mall vs street vs convenience store
- smartphone vs CCD vs mirrorless vs telephoto
- close-up vs full-body vs environment-heavy
- walking vs waiting vs sitting vs adjusting
- calm vs playful vs tired vs social

No repeated:
- same swimwear silhouette
- same cover layer
- same streetwear formula
- same camera angle
- same environment family
- same posture behavior
