## PHILOSOPHY

**Core Problem:** AI-generated images treat background as flat wallpaper behind the subject. Real photography has spatial depth — things exist at different distances from camera.

**The Fix:** Every image needs foreground / subject / background STRUCTURE that creates real spatial depth.

---

## DEPTH LAYER SYSTEM

### Three-Zone Spatial Model

```
┌─────────────────────────────────────┐
│  BACKGROUND    │  SUBJECT  │ FOREGROUND │
│  (far plane)   │ (mid)     │ (near)     │
│  40-60% frame  │ 30-40%    │ 5-15%      │
└─────────────────────────────────────┘
```

**Background:** Sets location, provides context, often soft/diffused
**Subject:** Clear focus, occupies mid-ground
**Foreground:** Creates depth, frames subject, adds realism

## SPATIAL DEPTH TRIGGERS

### FOREGROUND CREATION TECHNIQUES

| Technique | Example | When to Use |
|-----------|---------|-------------|
| Object at frame edge | Towel corner, shoe, bag | Beach, hotel, bedroom |
| Body part cropped close | Feet, hands, shoulder | Intimate, voyeur, documentary |
| Fabric in frame | Blanket fold, curtain | Indoor, domestic |
| Environmental edge | Door frame, window frame | Indoor/outdoor transition |
| Shadow as foreground | Shadow on floor | Any space with strong light |

### BACKGROUND SOFTNESS CONTROL

| Distance | Treatment | Example |
|----------|-----------|---------|
| Far background | Strong bokeh | City lights, ocean |
| Mid background | Soft focus | Building facade, wall |
| Near background | Sharp | Furniture, curtains, plants |

## SPATIAL COMPOSITION RULES

1. **Never flat background** — always 2-3 depth zones
2. **Foreground at 5-15% frame** — enough to create depth, not block subject
3. **Background provides context** — soft or sharp depending on mood
4. **Subject is clear focus point** — mid-ground by default
5. **Spatial logic must make sense** — objects at same depth behave similarly
